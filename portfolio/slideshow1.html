Happy + Hour(s)

HTML/CSS/Bootsrap/jQuery/Node/MongoDB

I created this app to help people find the happy hour...hours around them. I felt a good way would be for restaurants to be able to log-in and update their hours for users to see.

I used facebook and google authorization with passport that was then saved in my mongoDB. I also utilized the Zomato API to pull in places that I was able to search via zipcode or specific restuarant name, which when hours were added, would save into my own database to create my own API.

Overall it was a fun project and I look forward to adding more functionality in version 2.0
